## Usage

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```

### Gas Snapshots

```shell
$ forge snapshot
```

### Deploy

```shell
$ forge script script/DeployCore.s.sol:DeployCore --rpc-url <your_rpc_url> --broadcast
```

### Help

```shell
$ forge --help
```
